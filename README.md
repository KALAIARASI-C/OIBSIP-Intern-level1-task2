# Personal Portfolio Website – Kalaiarasi C

## Objective

The objective of this project is to design and develop a modern personal portfolio website to showcase my academic achievements, technical skills, internships, certifications, and projects. The website acts as an online professional profile that can be shared with recruiters and collaborators.

## Steps Performed

1. Planned the structure of the portfolio website including sections such as Hero, About, Skills, Internships, Certifications, Projects, and Achievements.
2. Created the basic webpage structure using **HTML**.
3. Styled the webpage using **CSS** to achieve a modern dark themed design.
4. Added icons for social media using **Font Awesome**.
5. Organized the content into different sections to improve readability and user experience.
6. Tested the website locally in the browser.
7. Uploaded the project files to **GitHub** and published it using **GitHub Pages**.

## Tools and Technologies Used

* HTML5
* CSS3
* Font Awesome (for social media icons)
* Visual Studio Code (Code Editor)
* GitHub (Code Hosting)
* GitHub Pages (Website Deployment)

## Outcome

The result of this project is a fully functional personal portfolio website that highlights my profile as a **B.Tech Information Technology student and aspiring Full Stack Developer**. The website presents my CGPA, technical skills, internships, certifications, and projects in a clean and responsive layout. It also provides links to my LinkedIn and GitHub profiles, making it easier for recruiters and collaborators to connect with me.
